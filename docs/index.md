## 2. TO-DO List

[![todo list app image][3]][4]

[3]: img/todolist.png
[4]: https://milan-micic.github.io/js-juniors/todoList  

TO-DO List je aplikacija za organizovanje zadataka koje sebi zadajemo.Prosto lista onog sto treba da uradimo.
